A responsive *Instagram Clone* built with React and Bootstrap.  
Implements Instagram-like UI features including feed, likes, comments, and profiles.  

---

## ✨ Features  
- 👤 User login/signup (demo only)  
- 🖼 Upload & view posts  
- ❤ Like and comment interactions  
- 🔄 Follow/unfollow (demo implementation)  
- 📱 Fully responsive mobile-first design  

---

## 🛠 Tech Stack  
- ⚛ React  
- 🎨 Bootstrap  
- 🌐 React Router
